# GPMSquare — Company Website

AI-based market research for Pharma & FMCG.

This is a static, single-file website (`index.html`) hosted via **GitHub Pages**.

## Deployment

GitHub Pages is already configured on this repo (Settings → Pages → Deploy from branch `main`, `/ (root)`).

To update the site:

1. Replace `index.html` in the repo root (upload via GitHub web UI or `git push`).
2. The `pages-build-deployment` workflow runs automatically.
3. The site refreshes at the GitHub Pages URL within a minute or two.

No build step, no dependencies — pure HTML/CSS with Google Fonts loaded from CDN.
